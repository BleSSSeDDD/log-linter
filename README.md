# log-linter

Линтер для проверки лог-записей в Go, совместимый с golangci-lint.

## Правила проверки

- Сообщения должны начинаться со строчной буквы
- Только английский язык
- Без спецсимволов и эмодзи
- Без чувствительных данных (пароли, токены)

## Установка и использование

```bash
# Сборка плагина
go build -buildmode=plugin -o loglinter.so plugin/plugin.go

# Запуск линтера
golangci-lint run

##Зависимости и требования
Go 1.22+

Поддерживаемые логгеры: log, slog
